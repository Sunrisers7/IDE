📝 КалькуляторЛёгкий
Задача: Напишите программу-калькулятор. Пользователь вводит два числа и операцию (+, -, *, /). Программа выводит результат. Обработайте деление на ноль.

Пример работы:
Первое число: 10
Второе число: 5
Операция: +
Результат: 15
📖 Показать решение 
a = float(input("Первое число: "))
b = float(input("Второе число: "))
op = input("Операция (+, -, *, /): ")

if op == "+":
    print(f"Результат: {a + b}")
elif op == "-":
    print(f"Результат: {a - b}")
elif op == "*":
    print(f"Результат: {a * b}")
elif op == "/":
    if b != 0:
        print(f"Результат: {a / b}")
    else:
        print("Ошибка: деление на ноль!")
else:
    print("Неизвестная операция")
🔢 FizzBuzzЛёгкий
Задача: Выведите числа от 1 до 100. Если число кратно 3 — выведите "Fizz", кратно 5 — "Buzz", кратно и 3 и 5 — "FizzBuzz". Иначе выведите само число.
📖 Показать решение 
for i in range(1, 101):
    if i % 3 == 0 and i % 5 == 0:
        print("FizzBuzz")
    elif i % 3 == 0:
        print("Fizz")
    elif i % 5 == 0:
        print("Buzz")
    else:
        print(i)
📂 Работа с файламиСредний
Задача: Напишите программу, которая создаёт файл numbers.txt, записывает в него числа от 1 до 10 (каждое с новой строки), затем читает файл и выводит сумму всех чисел.
📖 Показать решение 
# Запись в файл
with open('numbers.txt', 'w') as f:
    for i in range(1, 11):
        f.write(f"{i}\n")

# Чтение и суммирование
total = 0
with open('numbers.txt', 'r') as f:
    for line in f:
        total += int(line.strip())

print(f"Сумма чисел: {total}")
🔐 Генератор паролейСредний
Задача: Напишите функцию, которая генерирует случайный пароль длиной 12 символов. Пароль должен содержать заглавные и строчные буквы, цифры и спецсимволы.
📖 Показать решение 
import random
import string

def generate_password(length=12):
    chars = string.ascii_letters + string.digits + "!@#$%^&*"
    password = ''.join(random.choice(chars) for _ in range(length))
    return password

print(generate_password())
📷 Детектор лицСложный
Задача: Используя OpenCV, напишите программу, которая открывает веб-камеру, находит лица в кадре и обводит их зелёной рамкой. Выход по клавише 'q'.

Установка: pip install opencv-python
📖 Показать решение 
import cv2

face_cascade = cv2.CascadeClassifier(
    cv2.data.haarcascades + 'haarcascade_frontalface_default.xml'
)

cap = cv2.VideoCapture(0)

while True:
    ret, frame = cap.read()
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
    faces = face_cascade.detectMultiScale(gray, 1.1, 5)

    for (x, y, w, h) in faces:
        cv2.rectangle(frame, (x, y), (x+w, y+h), (0, 255, 0), 2)

    cv2.imshow('Face Detection', frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()

🔗 GitHub и деплой проектов

📖 Что такое Git и GitHub?

Git — система контроля версий. Сохраняет историю изменений кода, позволяет откатываться к предыдущим версиям и работать в команде.

GitHub — облачное хранилище для Git-репозиториев. Это как "Google Диск для программистов".

📦 Установка и настройка

  • Скачайте Git с git-scm.com и установите.
  • Настройте имя и email: 
    git config --global user.name "Ваше Имя"
git config --global user.email "your@email.com"
  • Зарегистрируйтесь на GitHub — github.com
